De qué trata

Nuestro Objetivo

Crear un prototipo de un sistema de publicación de datos abiertos basado en la tecnología blockchain. Los datos que se mostrarán provendrán de la toma de datos de sensores del entorno ambiental y climático que utilizan la tecnología LoRaWAN.

Implementación

Proceso

  • Diseño de la Arquitectura: Se definió el flujo de datos desde el sensor UV, pasando por el gateway LoRaWAN, una cola MQTT (Mosquitto), y su registro en la red blockchain con Hyperledger Fabric.

  • Implementación Técnica: Configuración del sensor, gateway, y broker MQTT; desarrollo de smart contracts para registrar los datos en la blockchain; y creación de una página web para visualización.

  • Pruebas y Validación: Verificación de la funcionalidad del sistema, pruebas de integridad en la blockchain, y optimización del flujo de datos.

  • Despliegue: Implementación en producción con monitoreo para garantizar estabilidad y rendimiento.

Recursos y/o Elementos

Se utilizaron sensores UV y un gateway LoRaWAN para capturar y transmitir datos. El backend incluye un broker MQTT (Mosquitto) y una red blockchain en Hyperledger Fabric.


Los datos se procesan y muestran en una página web desarrollada con frameworks como React o Vue.js, mientras que el sistema está alojado en servidores locales o en la nube, con contenedores Docker para la blockchain.
